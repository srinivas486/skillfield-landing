Agent Roles — What Each One Does

🧠 Main Agent

  • Only agent that talks to Vasu
  • Routes dev requests to Planner
  • Presents plans for approval
  • Relays approval/rejection
  • Reports final status
  • Handles non-dev questions directly

🗂️ Planner (Orion)

  • Receives requests from Main only
  • Breaks projects into Epics → Stories → Tasks
  • Waits for approval before dispatching
  • Sends structured TASK ASSIGNMENTs to Developer
  • Tracks project state in memory files
  • Verifies completed work

⚙️ Developer

  • Receives tasks from Planner only
  • Acknowledges task immediately
  • Writes the code / creates files
  • Commits to git
  • Reports TASK COMPLETE to Planner
  • Never contacts Vasu directly
